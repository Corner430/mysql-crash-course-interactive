📝 核心概念
──────────────────────────────────────

1️⃣  数据库 (Database)
   • 定义：保存有组织的数据的容器
   • 比喻：像一个文件柜，分类存储信息

2️⃣  表 (Table)
   • 定义：某种特定类型数据的结构化清单
   • 示例：顾客表、产品表、订单表
   • 规则：每个表有唯一的名字

3️⃣  列 (Column)
   • 定义：表中的一个字段
   • 示例：顾客编号、姓名、地址
   • 数据类型：每列都有特定的数据类型

4️⃣  行 (Row)
   • 定义：表中的一条记录
   • 示例：一个顾客的完整信息

5️⃣  主键 (Primary Key)
   • 定义：唯一标识表中每一行的列
   • 特性：唯一性、非空
   • 示例：顾客ID、产品ID

6️⃣  SQL (Structured Query Language)
   • 定义：结构化查询语言
   • 用途：与数据库通信的专用语言
   • 优点：简单、强大、通用

✨ 小结
──────────────────────────────────────

在本章中，你学习了：
• 数据库是一个有组织的数据容器
• 表是存储特定类型数据的结构
• 列定义了存储的数据类型
• 行是表中的具体记录
• 主键唯一标识每一行
• SQL 是与数据库通信的语言

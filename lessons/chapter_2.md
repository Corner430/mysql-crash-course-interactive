📝 核心知识点
──────────────────────────────────────

1️⃣  什么是 MySQL
   
   MySQL是一个**数据库管理系统(DBMS)**
   
   特点：
   • 开源免费
   • 性能优秀
   • 可靠性高
   • 简单易用
   • 跨平台支持


2️⃣  客户机-服务器架构
   ────────────────────────────────────
   MySQL采用客户机-服务器模式：
   
   ┌──────────┐         ┌──────────┐
   │  客户端  │ ←────→ │  服务器  │
   │ (Client) │  网络  │ (Server) │
   └──────────┘         └──────────┘
   
   工作原理：
   1. 服务器端运行MySQL DBMS
   2. 客户端发送SQL请求
   3. 服务器处理请求并返回结果
   
   优势：
   • 数据集中管理
   • 多用户同时访问
   • 安全控制更好
   • 性能优化更容易


3️⃣  MySQL 版本
   ────────────────────────────────────
   主要版本：
   • MySQL 4.x - 老版本
   • MySQL 5.x - 广泛使用
   • MySQL 8.x - 最新版本
   
   选择建议：
   • 生产环境：使用稳定的LTS版本
   • 学习练习：任何5.x以上版本都可以

🔧 MySQL 工具
──────────────────────────────────────

工具 1: mysql 命令行实用程序
   ────────────────────────────────────
   最常用的MySQL客户端工具
   
   连接方式：
   mysql -h主机名 -u用户名 -p密码 数据库名
   
   示例：
   docker exec -it mysql mysql -uroot -ppassword
   
   常用命令：
   • help 或 \h - 帮助
   • quit 或 exit - 退出
   • \c - 取消当前命令
   • status - 查看服务器状态


工具 2: MySQL Workbench (图形化工具)
   ────────────────────────────────────
   官方提供的图形化管理工具
   
   功能：
   • 可视化数据库设计
   • SQL编辑器
   • 服务器管理
   • 性能监控
   
   下载：https://dev.mysql.com/downloads/workbench/


工具 3: phpMyAdmin (Web界面)
   ────────────────────────────────────
   基于Web的MySQL管理工具
   
   特点：
   • 通过浏览器访问
   • 操作简单直观
   • 适合快速管理


工具 4: 其他第三方工具
   ────────────────────────────────────
   • Navicat - 商业软件，功能强大
   • DBeaver - 免费开源，支持多种数据库
   • HeidiSQL - Windows下轻量级工具

💡 重要提示
──────────────────────────────────────

⚠️  客户机-服务器模式理解
    
    • 服务器：MySQL DBMS软件，负责数据存储和处理
    • 客户端：连接到服务器的工具（mysql、Workbench等）
    • 一个服务器可以同时服务多个客户端


⚠️  命令行使用技巧
    
    • 所有SQL语句以分号(;)结束
    • 使用\c可以取消正在输入的命令
    • 上下箭头可以浏览命令历史
    • Tab键可以自动补全（部分版本）


⚠️  密码安全
    
    在命令行中直接输入密码会有安全警告：
    mysql: [Warning] Using a password on the command line 
           interface can be insecure.
    
    这是正常的，学习环境可以忽略
    生产环境建议使用配置文件存储密码


⚠️  版本差异
    
    不同版本的MySQL在某些功能上有差异
    本教程基于MySQL 5.x，但大部分内容适用于8.x

📊 实战示例
──────────────────────────────────────

示例 1: 完整的连接流程
  
  # 1. 连接到MySQL
  docker exec -it mysql mysql -uroot -ppassword
  
  # 2. 查看版本
  mysql> SELECT VERSION();
  
  # 3. 查看当前用户
  mysql> SELECT USER();
  
  # 4. 查看所有数据库
  mysql> SHOW DATABASES;
  
  # 5. 退出
  mysql> exit;


示例 2: 查看服务器信息
  
  mysql> status;
  
  输出类似：
  --------------
  mysql  Ver 8.0.x for Linux
  
  Connection id:          10
  Current database:       
  Current user:           root@localhost
  ...


示例 3: 使用help命令
  
  mysql> help
  
  查看特定命令的帮助：
  mysql> help SELECT;
  mysql> help CREATE TABLE;

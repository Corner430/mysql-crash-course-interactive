# 🚀 MySQL 必知必会 - 快速开始指南

欢迎使用 MySQL 交互式学习系统！本系统包含**完整30章**内容，覆盖《MySQL必知必会》全书。

## ⚡ 60秒快速开始

### 步骤1: 初始化数据库（首次使用）

```bash
docker exec -i mysql mysql -uroot -ppassword < setup_db.sql
```

✅ 成功后会创建 `crashcourse` 数据库和6个示例表

### 步骤2: 启动学习系统

```bash
./mysql_learn.sh
```

### 步骤3: 选择章节

在菜单中输入章节编号（1-30）开始学习

### 步骤4: 开始实践

选择 `1. 进入 MySQL 实践` 直接操作数据库

## 🚀 快速开始

### 前置要求

- Docker 和 Docker Compose
- Linux/macOS/Windows (支持 Bash)

### 1. 启动 MySQL 环境

**方式一：使用 Docker Compose（推荐）**

```bash
# 启动 MySQL 容器
docker-compose up -d

# 等待 MySQL 启动完成（约 10-20 秒）
docker-compose ps
```

**方式二：手动启动 Docker 容器**

```bash
docker run -d \
  --name mysql \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=password \
  -e MYSQL_DATABASE=crashcourse \
  mysql:8.0
```

### 2. 初始化数据库

```bash
# 导入示例数据库
docker exec -i mysql mysql -uroot -ppassword < setup_db.sql
```

或在学习脚本中选择 `0. 初始化数据库`

### 3. 启动学习系统

```bash
./mysql_learn.sh
```

### 4. 选择章节学习

从主菜单中选择章节编号（1-30）开始学习

### 5. 进入实践模式

在章节页面选择 `1. 进入 MySQL 实践` 直接操作数据库

### ⚠️ 安全说明

本项目使用默认密码 `password` 仅用于**本地学习环境**：
- ✅ 适用于本地 Docker 容器学习
- ❌ **请勿在生产环境使用此密码**
- 🔒 如需修改密码，请参考下方"修改配置"说明

### 修改配置

如需修改密码或其他配置：

1. 编辑 `docker-compose.yml` 修改 `MYSQL_ROOT_PASSWORD`
2. 编辑 `mysql_learn.sh` 中的 `MYSQL_PASS` 变量（第23行）
3. 编辑 `test_system.sh` 中的 `MYSQL_PASS` 变量（第6行）

### 停止和清理

```bash
# 停止容器
docker-compose down

# 停止并删除数据卷（清空所有数据）
docker-compose down -v
```
